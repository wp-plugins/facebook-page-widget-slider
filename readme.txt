Facebbook Page Plugin allows to promote your facebook page under your website. 

== Description ==

Placing a facebook widget like this on website is always been tricky and confusing. By using facebook page widget slider you can easily put your facebook plugin on your website and convert your website visits to revenue.

Facebook Page Widget Slider put a facebook icon on your website absolute left or right position. Using Facebook Page Widget Slider you can easily make sure the page attract your website visitors.

Just like on Facebook, your visitors can like and share the Page without having to leave your site.

DEMO - http://connexstudios.com/wpdemo

== Installation ==

Install the Plugin - top

Login to your wordpress admin - example : http://www.domain.com/wp-admin
go to "Plugins->Installed Plugins" from Menu
Click "Add New" - button at top
Click "Upload" - from the list
Click "Choose File" - button
a windows will open of your computer - browse file - select the package
click "Install Now"

On successful completion of installation you will get a success massage.

Now you have to activate the plugin.

You are all set - Now if you go to your website homepage you probably will see the sidebar up with default options.

To customize the facebook page plugin you have to change the settings.

From Wordpress ADMIN area - go to -> Settings > Facebook Page Widget Slider

Edit the options available as your preferences. Once done close everything and see the plugin on your website.
